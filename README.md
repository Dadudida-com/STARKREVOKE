## Getting Started

rename `.env.example` as `.env` and set fields

```bash
ALCHEMY_URL='https://starknet-mainnet.g.alchemy.com/v2/<API_KEY>'
API_KEY = '<YOUR STARKSCAN API KEY>'
```
Then install dependencies 
```bash
yarn install
```

Ready to run the development server:

```bash
yarn dev
```
